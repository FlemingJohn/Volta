# IEC 60228: Conductors of Insulated Cables

## 1. Scope
**IEC 60228** specifies the **nominal cross-sectional areas** and requirements for conductors in electric power cables.
* **Key Concept:** "Nominal Cross-Section" is a *label*, not a strict measurement. A conductor is compliant if it meets the **Maximum DC Resistance** requirement, regardless of its exact geometric diameter.

---

## 2. Conductor Classes
The standard defines four main classes based on flexibility.

### Class 1: Solid Conductors
* **Description:** Single solid wire of copper or aluminum.
* **Marking:** Often denoted as `(re)` for round solid or `(se)` for sector solid.
* **Application:** Fixed installations, low cost, rigid.
* **Material:** Plain Copper, Metal-coated Copper, Aluminum.

### Class 2: Stranded Conductors
* **Description:** A number of individual wires stranded together.
* **Marking:** `(rm)` for round stranded, `(sm)` for sector stranded.
* **Application:** Fixed power cables (most common for industrial power).
* **Compaction:** Can be compacted to reduce diameter.

### Class 5: Flexible Conductors
* **Description:** Many fine wires.
* **Application:** Cords, household appliances, portable equipment.
* **Requirement:** Max diameter of component wires is restricted (e.g., 0.21mm - 0.41mm).

### Class 6: Extra Flexible Conductors
* **Description:** Very fine wires.
* **Application:** Welding cables, robotics, moving machinery.
* **Requirement:** Component wires are finer than Class 5 (e.g., 0.16mm).

## 5. Temperature Correction Formula
The standard values are at **20°C**. For other temperatures ($t$), calculate resistance ($R_t$) using:
$$R_t = R_{20} \times [ 1 + \alpha_{20} (t - 20) ]$$
* **$\alpha_{20}$ (Copper):** 0.00393 per °C
* **$\alpha_{20}$ (Aluminum):** 0.00403 per °C
